## üöÄ Getting Started  

To get started, clone the repository:  
```sh
git clone https://github.com/dubewarsagar/calculator-docs.git
cd calculator-docs
```

To build the documentation using **DocFx**, run:  
```sh
docfx docfx.json
```

Then, open `_site/index.html` in your browser.
